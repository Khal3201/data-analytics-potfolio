# Report 02 - Most Rented Movies Analysis

## Description

This report identifies the most rented movies using the **Sakila** sample database provided by MySQL.

The purpose is to determine which movies are the most popular among customers, providing useful information for inventory management and business decision-making.

---

## Objective

Identify the movies with the highest number of rentals.

---

## Business Question

> Which movies are rented most frequently by customers?
