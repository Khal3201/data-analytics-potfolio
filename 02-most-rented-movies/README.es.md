# Reporte 02 - Análisis de las películas más rentadas

## Descripción

En este análisis se identifican las películas con mayor número de rentas utilizando la base de datos de ejemplo **Sakila** de MySQL.

El objetivo es obtener un ranking de las películas más populares para generar información útil para la toma de decisiones relacionadas con inventario, disponibilidad de copias y estrategias comerciales.

---

## Objetivo

Determinar cuáles son las películas con mayor demanda por parte de los clientes.

---

## Pregunta de negocio

> ¿Cuáles son las películas más rentadas por los clientes?
